This program reminds you of people's birthdays.

In order to run this program, you need to edit some lines of code.

1 - Update the birthdays.csv with your friends & family's details. 
2- Update MY_EMAIL and PASSWORD with your proper mail data
2.1 - Your mail must have "2-step verification" and through this, you can create a new app password. 
      After that, you should use the given password instead of your actual real password.
